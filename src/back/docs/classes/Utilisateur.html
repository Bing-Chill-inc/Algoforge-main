Class Utilisateur

Modèle de données pour les utilisateurs. Un utilisateur peut avoir aucun ou plusieurs tokens. Un utilisateur peut avoir aucun ou plusieurs permissions sur les dossiers. Un utilisateur peut avoir aucun ou plusieurs permissions sur les algorithmes.
